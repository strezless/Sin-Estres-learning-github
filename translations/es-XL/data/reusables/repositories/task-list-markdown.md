Para crear una lista de tareas, introduce los elementos de la lista con un caracter de espacio regular, seguido de `[ ]`. Para marcar una tarea como completada, utiliza `[x]`.

```
- [x] Finish my changes
- [ ] Push my commits to GitHub
- [ ] Open a pull request
```
