| キー       | 種類       | 説明                                                           |
| -------- | -------- | ------------------------------------------------------------ |
| `action` | `string` | 実行されたアクション. Can be `added` or `removed`.                     |

| `member` | `オブジェクト` | The [user](/rest/reference/users) that was added or removed. |
| `Team`   | `オブジェクト` | The [team](/rest/reference/teams) for the membership.        |
