コミットをコマンドラインからプッシュする際には、[Git で設定した](/articles/setting-your-commit-email-address)メールアドレスがコミットに関連付けられます。 この設定により、コマンドライン上で個人のメールアドレスを使用してコミットをプッシュすることがブロックされます。
