[GitHub Jobs](https://jobs.github.com/) ボードは、テック業界の仕事を探す優れた方法です。 GitHub ダッシュボードにポストされた仕事を見るために選択できます。

![ダッシュボードの GitHub Jobs 広告](/assets/images/help/settings/jobs-ads-on-dashboard.png)
