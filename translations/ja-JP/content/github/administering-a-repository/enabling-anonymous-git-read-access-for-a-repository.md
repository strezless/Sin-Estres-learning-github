次の場合に、リポジトリの管理者は、特定のリポジトリに対する匿名 Git 読み取りアクセスの設定を変更できます。
- サイトの管理者がプライベートモードと匿名 Git 読み取りアクセスを有効化している。
- The repository is public on the enterprise and is not a fork.
- サイト管理者がリポジトリで匿名 Git 読み取りアクセスを無効化していない。
