個人アカウントに対する請求書の対応や、発注書のサポートはしておりません。 アカウントの月次もしくは年次の支払日に、領収書をメールします。 あなたの会社、国、会計士が領収書にさらなる詳細を必要とする場合は、領収書に[情報を追加](/articles/adding-information-to-your-personal-account-s-receipts)できます。
