1. タグに署名するには、`git tag` コマンドに `-s` を追加します。
  ```shell
  $ git tag -s <em>mytag</em>
  # 署名済みのタグを作成する
  ```
2. `git tag -v [tag-name]`を実行して、署名したタグをベリファイします。
  ```shell
  $ git tag -v <em>mytag</em>
  # 署名済みのタグを検証する
  ```
