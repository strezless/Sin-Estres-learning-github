Os administradores de repositório podem habilitar proteções em um branch. Se estiver trabalhando em um branch que é protegido, não será possível excluir nem forçar o push no branch. Os administradores de repositório podem ativar outras configurações protegidas de branch para aplicar fluxos de trabalho específicos antes que um branch possa ser mesclado. Para obter mais informações, consulte "[Sobre branches protegidos](/articles/about-protected-branches)".
