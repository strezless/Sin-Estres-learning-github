Se as referências de link automático personalizado estão configuradas para um repositório, referências a recursos externos, como um problema do JIRA ou um ticket do Zendesk, serão convertidas em links encurtados. Para saber quais links automáticos estão disponíveis no repositório, entre em contato com alguém com permissões de administrador no repositório. Para obter mais informações, consulte "[Configurar links automáticos para fazer referência a recursos externos](/articles/configuring-autolinks-to-reference-external-resources)".
