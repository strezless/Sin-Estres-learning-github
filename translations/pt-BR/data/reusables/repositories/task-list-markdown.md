Para criar uma lista de tarefas, introduza os itens da lista com um caractere de espaço normal seguido de `[ ]`. Para marcar uma tarefa como concluída, use `[x]`.

```
- [x] Finish my changes
- [ ] Push my commits to GitHub
- [ ] Open a pull request
```
