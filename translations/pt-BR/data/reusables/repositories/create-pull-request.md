1. Para criar um pull request que está pronto para revisão, clique em **Criar Pull Request**. Para criar um rascunho de pull request, use o menu suspenso e selecione **Criar rascunho de pull request** e, em seguida, clique em **Rascunho de Pull Request**. Para obter mais informações sobre pull requests em rascunho, consulte "[Sobre pull requests](/articles/about-pull-requests#draft-pull-requests)".![Botão de Criar pull request](/assets/images/help/pull_requests/pullrequest-send.png)
