**Nota:** Quando você remover um usuário da sua organização, os dados de associação deles ficam salvos por três meses. É possível restaurar os dados deles ou quaisquer bifurcações privadas que pertençam aos repositórios da organização, se você convidar o usuário para se juntar novamente à organização dentro desse período. Para obter mais informações, consulte "[Restabelecer ex-integrantes da organização](/articles/reinstating-a-former-member-of-your-organization)".
