2. Klicke oben im Kommentar auf **edited** (bearbeitet) um den Änderungsverlauf zu öffnen.

      {% note %}

      **Hinweis:** Wenn die **edited** (bearbeitet) Option nicht siehst, hat der Kommentar keinen Änderungsverlauf.

      {% endnote %}
