1. Um Deinen GPG-Signaturschlüssel in Git festzulegen, fügst Du den nachfolgenden Text ein und ersetzt die GPG-Schlüssel-ID, die Du verwenden möchtest. Im folgenden Beispiel lautet die GPG-Schlüssel-ID `3AA5C34371567BD2`:
  ```shell
$ git config --global user.signingkey <em>3AA5C34371567BD2</em>
```
