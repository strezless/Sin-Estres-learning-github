Ein externer Mitarbeiter, beispielsweise ein Berater oder ein vorübergehend Beschäftigter, ist eine Person, die auf mindestens ein Organisations-Repository zugreifen kann, jedoch kein explizites Mitglied der Organisation ist. Weitere Informationen findest Du unter:
