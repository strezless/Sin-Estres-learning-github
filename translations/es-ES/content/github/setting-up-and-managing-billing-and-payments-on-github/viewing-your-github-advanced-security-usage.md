Cada licencia de {% data variables.product.prodname_GH_advanced_security %} especifica una cantidad máxima de cuentas o de plazas que pueden utilizar estas características. Debes revisar de vez en cuando que tu uso esté dentro de la capacidad de tu licencia. Para obtener más información, consulta la sección "[Acerca del licenciamiento para la licencia empresarial de {% data variables.product.prodname_GH_advanced_security %}](/github/setting-up-and-managing-billing-and-payments-on-github/about-licensing-for-github-advanced-security)".
