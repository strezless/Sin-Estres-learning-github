#### Parameters

| Name           | Тип      | Description                                                    |
| -------------- | -------- | -------------------------------------------------------------- |
| `access_token` | `строка` | The OAuth access token used to authenticate to the GitHub API. |

#### Пример

```json
{
  "access_token": "e72e16c7e42f292c6912e7710c838347ae178b4a"
}
```
