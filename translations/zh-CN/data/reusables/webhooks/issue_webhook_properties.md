| 键        | 类型    | 描述                                                                                                                                                                                                   |
| -------- | ----- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `action` | `字符串` | 执行的操作内容. 可以是以下项之一：`opened`、`edited`、`deleted`、`pinned`、`unpinned`、`closed`、`reopened`、`assigned`、`unassigned`、`labeled`、`unlabeled`、`locked`、`unlocked`、`transferred`、`milestoned` 或 `demilestoned`。 |
