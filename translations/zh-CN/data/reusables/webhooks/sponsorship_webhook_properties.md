| 键        | 类型    | 描述                                                                                                              |
| -------- | ----- | --------------------------------------------------------------------------------------------------------------- |
| `action` | `字符串` | 执行的操作内容. 可以是以下项之一：`created`、`cancelled`、`edited`、`tier_changed`、`pending_cancellation` 或 `pending_tier_change`。 |
