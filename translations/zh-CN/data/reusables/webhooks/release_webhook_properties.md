| 键        | 类型    | 描述                                                                                                                                                                                                                          |
| -------- | ----- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `action` | `字符串` | 执行的操作内容. 可以是以下选项之一：<ul><li>`published`：发行版、预发行版或发行版草稿已发布</li><li>`unpublished`：发行版或预发行版已删除</li><li>`created`：草稿已保存，或者此前未保存为草稿的发行版或预发行版已发布</li><li>`edited`：发行版、预发行版或发行版草稿被编辑</li><li>`deleted`：发行版、预发行版或发行版草稿被删除</li><li>`prereleased`：预发行版已创建</li><li>`released`：发行版或发行版草稿已发布，或者预发行版被更改为发行版</li> |
