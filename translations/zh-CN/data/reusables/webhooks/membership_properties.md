| 键        | 类型    | 描述                                  |
| -------- | ----- | ----------------------------------- |
| `action` | `字符串` | 执行的操作内容. 可以是 `added` 或 `removed`。   |
| `作用域`    | `字符串` | 成员的作用域。 目前只能是 `team`。               |
| `成员`     | `对象`  | 已添加或删除的[用户](/rest/reference/users)。 |
| `团队`     | `对象`  | 成员的[团队](/rest/reference/teams)。     |
