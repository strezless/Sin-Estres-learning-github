| 键        | 类型    | 描述                                                                                                                                                        |
| -------- | ----- | --------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `action` | `字符串` | 执行的操作内容. 可以是以下项之一：`opened`、`closed`、`reopened`、`assigned`、`unassigned`、`review_requested`、`review_request_removed`、`labeled`、`unlabeled` 和 `synchronize`。 |
