| 键              | 类型    | 描述                                                                                                                  |
| -------------- | ----- | ------------------------------------------------------------------------------------------------------------------- |
| `action`       | `字符串` | 执行的操作内容. 可以是 `requested` 或 `completed` 之一。                                                                          |
| `workflow_run` | `对象`  | 工作流程运行。 许多 `workflow_run` 键，如 `head_branch`、`conclusion` 和 `pull_requests`，与 [`check_suite`](#check_suite) 对象中的键相同。 |
