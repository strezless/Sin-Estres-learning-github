**警告：**永远不要 `git 添加`、`提交`或`推送`敏感信息到远程仓库。 敏感信息包括但不限于：

- 密码
- SSH keys
- [AWS 访问密钥](http://docs.aws.amazon.com/AWSSimpleQueueService/latest/SQSGettingStartedGuide/AWSCredentials.html)
- API 密钥
- 信用卡号码
- PIN 码

更多信息请参阅“[从仓库中删除敏感数据](/articles/removing-sensitive-data-from-a-repository)”。

{% endwarning %}
