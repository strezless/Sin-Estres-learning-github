# GraphQL

`/content/graphql` 目录是 GitHub GraphQL API 文档所在的位置！

* `/content/graphql/guide` and `/content/graphql/overview` 目录中包含可以编辑的项目。
* `/content/graphql/references` 目录对 GitHub GraphQL API 中使用的每个GraphQL 数据类型包含一个项目。 此目录中的大多数内容都使用 `include` 呈现。

  `include` 标记呈现的内容来自 `/lib/graphql/static` 目录，由 GitHub 内部的 API 源代码自动生成，不应人工编辑。 更多信息请参阅 [`/lib/graphql/README.md`](/lib/graphql/README.md)。

  **因此，我们不能接受参与此仓库中的 GraphQL API 参考内容。**
