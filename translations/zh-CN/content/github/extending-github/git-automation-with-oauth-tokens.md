### 第 1 步：获取 OAuth 令牌

在应用程序设置页面上创建个人访问令牌。 更多信息请参阅“[创建个人访问令牌](/github/authenticating-to-github/creating-a-personal-access-token)”。

{% warning %}

**警告**：令牌具有读取/写入权限，应该被视为密码。 如果您在克隆或添加远程仓库时将令牌输入克隆 URL，Git 会以纯文本格式将其写入 _.git/config_ 文件，这存在安全风险。

{% endwarning %}
