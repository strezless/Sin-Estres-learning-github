您可以按以下各项排序任何过滤的视图：

* 最新创建的议题或拉取请求
* 最早创建的议题或拉取请求
* 评论最多的议题或拉取请求
* 评论最少的议题或拉取请求
* 最新更新的议题或拉取请求
* 最早更新的议题或拉取请求
* 添加[反应](/articles/about-conversations-on-github#reacting-to-ideas-in-comments)最多的议题或拉取请求
