`-K` 选项位于 Apple 的 `ssh-add` 标准版本中，当您将 ssh 密钥添加到 ssh-agent 时，它会将密码存储在您的密钥链中。 如果您已安装不同版本的 `ssh-add`，该版本可能缺少对 `-K` 的支持。

### 解决问题

要将 SSH 私钥添加到 ssh 代理，可以指定到 `ssh-add` Apple 版本的路径：

```shell
  $ /usr/bin/ssh-add -K ~/.ssh/id_rsa
```
