如果提交合作作者的提交已合并到仓库的默认分支并且他们是贡献最多提交的前 15 名用户，则提交合作作者将包括在提交活动摘要中。

### 按时间过滤

默认情况下，Pulse 显示过去七天的仓库活动。 要选择不同的时间段，请单击 Pulse 概览右上角中的 **Period（周期）**下拉列表。
