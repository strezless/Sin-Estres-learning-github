{% tip %}

**提示：**

- 文件查找器的结果不包括某些目录，例如 `build`、`log`、`tmp` 和 `vendor`。 要搜索这些目录中的文件，请使用 [`filename` 代码搜索限定符](/articles/searching-code#search-by-filename)。
- 按键盘上的 `t` 键也可以打开文件查找器。 更多信息请参阅“[键盘快捷键](/articles/keyboard-shortcuts)”。

{% endtip %}

4. 在结果列表中 ，单击要查找的文件。
